# SmartBattleSpeed

《龙胤立志传》智能战斗倍速 Mod。

战斗太慢容易坐牢，直接全局 10 倍又容易错过主角操作和关键战斗表现。本 Mod 可以分别控制主角行动、NPC 行动以及 NPC 攻击主角时的战斗速度，在加快战斗节奏的同时保留需要观察和操作的部分。

## 功能

- 在战斗倍速栏增加“智能”和“设置”按钮。
- 主角行动与 NPC 行动可以分别设置倍速。
- NPC 攻击主角时可以使用单独的倍速。
- 支持 `1x`、`2x`、`3x`、`5x`、`10x` 五档速度。
- 战斗中修改设置后即时生效。
- 智能模式和倍速设置可以跨游戏启动保存。
- 点击游戏原生倍速按钮时自动退出智能模式，恢复原生倍速逻辑。
- 玩家不上场的战斗中自动调整“跳过”按钮位置，避免遮挡 Mod 按钮。

例如将主角设为 `1x`、NPC 设为 `10x`、NPC 攻击主角设为 `1x`，普通 NPC 行动会快速通过，轮到主角或 NPC 正在攻击主角时则恢复正常速度。

## 安装

本 Mod 需要 [MelonLoader](https://melonwiki.xyz/)。

1. 为《龙胤立志传》安装 MelonLoader。
2. 运行一次游戏，确认游戏目录中已生成 `Mods` 文件夹。
3. 将 `SmartBattleSpeedMod.dll` 放入游戏目录的 `Mods` 文件夹。
4. 启动游戏。

安装路径示例：

```text
LongYinLiZhiZhuan\Mods\SmartBattleSpeedMod.dll
```

## 使用

进入战斗后：

- 点击“智能”开启智能战斗倍速。
- 点击“设置”打开倍速设置面板。
- 点击原生 `x1`、`x2`、`x3`、`x5` 或 `x10` 按钮，退出智能模式并切回游戏原生倍速。

设置面板可以分别选择：

- 主角倍速：`1x` / `2x` / `3x` / `5x` / `10x`
- NPC 倍速：`1x` / `2x` / `3x` / `5x` / `10x`
- NPC 攻击主角倍速：`1x` / `2x` / `3x` / `5x` / `10x`

## 注意事项

- 只有点击“智能”后，Mod 才会接管战斗倍速。
- 点击游戏原生倍速按钮会退出智能模式。
- 保存设置后，只有当前处于智能模式时才会立即应用到本场战斗。
- 反馈问题时请附上操作方式、当前设置、战斗界面和 MelonLoader 控制台报错截图。
